## Install

Download from [status302](http://repo.status302.com/browse/afIocEnv).

Or install via fanr:

    $ fanr install -r http://repo.status302.com/fanr/ afIocEnv

To use in a project, add a dependency in your `build.fan`:

    depends = ["sys 1.0", ..., "afIocEnv 0+"]
